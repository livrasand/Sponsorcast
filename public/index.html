Quick Start

Embed protected videos in just three lines of code:

<!-- Include required scripts --> <script src="https://cdn.jsdelivr.net/npm/hls.js@latest"></script> <script src="https://your-domain.com/sponsorcast.js"></script> <!-- Embed protected video --> <sponsor-cast src="your-video-id" github-user="your-github-username" width="720" autoplay> </sponsor-cast>

Component Attributes

Attribute Type Description
src required string Video ID or folder name containing HLS segments
github-user required string GitHub username to check sponsorship for
width optional string Video width (default: 720px)
height optional string Video height (default: auto)
autoplay optional boolean Enable autoplay (muted by default)

Setup Instructions

  1. Deploy to Vercel
    Fork the repository and deploy to Vercel with one click
  2. Create GitHub OAuth App
    Set up a new OAuth application in your GitHub developer settings
  3. Configure Environment Variables
    Add the following variables to your Vercel deployment:
    GITHUB_CLIENT_ID=your_oauth_app_id GITHUB_CLIENT_SECRET=your_oauth_app_secret GITHUB_USER=your_username JWT_SECRET=random_secret_string HOST=https://your-domain.com
  4. Upload Videos
    Use sponsorcast-cli or manually upload HLS files to the videos folder
