Quick Start

Embed protected videos in just three lines of code:

<!-- Include required scripts --> <script src="https://cdn.jsdelivr.net/npm/hls.js@latest"></script> <script src="https://sponsorcast.vercel.app/sponsorcast.js"></script> <!-- Embed protected video --> <sponsor-cast src="your-video-id" github-user="your-github-username" width="720" autoplay> </sponsor-cast>

Component Attributes

Attribute Type Description
src required string Video ID or folder name containing HLS segments
github-user required string GitHub username to check sponsorship for
width optional string Video width (default: 720px)
height optional string Video height (default: auto)
autoplay optional boolean Enable autoplay (muted by default)

Setup Instructions

  1. 1. Sign Up for Sponsorcast
    Create your free account at sponsorcast.vercel.app. You'll connect your GitHub account and select the user or organization you want to link with Sponsorcast.
  2. 2. Install Sponsorcast CLI
    Make sure you have Node.js 16+ and FFmpeg installed. Then install the CLI globally:
    npm install -g sponsorcast-cli
  3. 3. Upload Your Video
    Use the CLI to convert your video to HLS format and upload it to Sponsorcast. Example:
    sponsorcast upload --video ./my-video.mp4
    Optionally add a custom title and thumbnail:
    sponsorcast upload --video ./my-video.mp4 --name "My GitHub Tutorial" --thumbnail ./thumb.jpg
  4. 4. Embed on Your Website
    Just copy and paste this into your site. Your video will only be visible to verified GitHub Sponsors.
